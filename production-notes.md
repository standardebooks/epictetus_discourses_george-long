(1) I made a lot of changes to the endnotes: expanding abbreviations and abbreviated names, cross-linking, etc.

(2) In the main text, Epictetus uses imaginary dialog a lot (arguing with an imaginary opponent); this can be difficult to read in Long's rendering, which just runs the sentences all together without showing who's talking. I've frequently added quotation marks to try to make this clearer.

(3) I've Americanized and modernized the English.

(4) The punctuation in the original is often an obstacle to understanding, and I've altered this when I thought it would help.

(5) I added an editorial note to endnote #528.

(6) In two cases I added empty `<span>` elements when a new paragraph followed a `<blockquote>` (that is, the `<blockquote>` did not interrupt the paragraph in the middle) so that the new paragraph would get indented properly.
